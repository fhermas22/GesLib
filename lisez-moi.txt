# Comment exécuter ce projet de Système de Gestion de Bibliothèque

1. Téléchargez et décompressez le fichier sur la bibliothèque de copie de votre système local.
2. Placez le dossier de la bibliothèque dans le répertoire racine

# Configuration de la base de données

Ouvrez phpmyadmin
Créer une bibliothèque de base de données
Importer la base de données library.sql (disponible dans le package zip)
